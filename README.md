# Learning_Python_Day3

Today's topic is using Python as a calculator!

**Goal:** Use Python as a calculator and build programs that perform mathematical operations

- **task1:** Calculate area of a rectangle (length * width)

- **task2:** Calculate circle area (3.14 * radius * 2)

- **task3:** Check if a number is even using % 2

- **task4:** Calculate average of three numbers

- **task5:** Use // and % to convert minutes into hours and remaining minutes

**mini-project:**
Bill Splitter: Ask for total bill amount, number of people, and tip percentage. Calculate tip amount, total with tip, and amount per person. Display all calculations clearly.
